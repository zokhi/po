Теги Описание Синтаксис Пример
body представляет собой контент (содержимое) документа HTML. В документе может быть только один элемент
h1 Тег H1 — это HTML-заголовок, который чаще всего используется для разметки заголовка веб-страницы.
ul элемент «Неупорядоченный список» HTML-элемент используется для неупорядоченного списка - в частности для маркированного списка.
ol Для создания нумерованного списка в HTML используется тег ol (ol сокращение от англ. ordered list - упорядоченный список),который информирует браузер,что все его содержимое,вплоть до закрывающего тега,представляет собой нумерованный список.
link предназначен для подключения внешних файлов (шрифтов или стилей)
img Элемент img (от англ. image — изображение) предназначен для отображения на веб-странице изображений в графическом формате GIF, JPEG, SVG или PNG
table Элемент table служит контейнером для элементов, определяющих содержимое таблицы
tr tr служит контейнером для создания строки таблицы.
td Предназначен для создания одной ячейки таблицы. Тег td должен размещаться внутри контейнера tr, который в свою очередь располагается внутри тега table.
head предназначен для хранения других элементов, цель которых — помочь браузеру в работе с данными
label устанавливает связь между определенной меткой, в качестве которой обычно выступает текст, и элементом формы (input select,textarea).Такая связь необходима, чтобы изменять значения элементов формы при нажатии курсором мыши на текст
tfoot Элемент tfoot предназначен для хранения одной или нескольких строк, которые представлены внизу таблицы. Хотя тег tfoot в исходном коде должен быть определен до тега tbody,браузеры отображают его в самом низу таблицы.
html html является контейнером, который заключает в себе все содержимое веб-страницы
